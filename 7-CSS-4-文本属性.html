CSS Text（文本）属性可定义文本的外观，比如文本的颜色、对齐文本、装饰文本、文本缩进、行间距等

文本颜色：color属性

表示方式属性值
预定义的颜色值red,green,blue,pink
十六进制(最常用)#FF0000,#FF6600,#29D794
RGB代码rgb(255,0,0)或rgb(100%,0%,0%)

对齐文本：text-align属性用于设置元素内文本内容的水平对齐方式。

div { text-align: center; }

属性值解释
left左对齐(默认值)
right右对齐
center居中对齐

装饰文本：text-decoration属性规定添加到文本的修饰。可以给文本添加下划线、删除线、上划线等

div { text-decoration: underline; }

属性值解释
none默认。没有装饰线（最常用）
underline下划线。链接a自带下划线（常用）
overline上划线。（几乎不用）
line-through删除线（不常用）

文本缩进：tect-indent属性用来指定文本的第一行的缩进，通常是将段落的首行缩进

div { text-indent: 10px; } 由此，所有元素的第一行都可以缩进一个给定的长度，该长度也可以为负值

div { text-indent: 2em; } em是一个相对单位，就是当前元素（font-size）1个文字的大小。eg：当前文字大小为16px，则1 em相当于16px，2em相当于32px。如果当前文字没有设置大小，则会按照父元素的1个文字大小。

行间距：text-height属性用于设置行间的距离（行高）。可以控制文字行与行之间的距离

p { line-height: 26px; }

行间距包括：上间距、文本高度、下间距。因此，行高改的其实是上间距和下间距。行高测量方式：上一行的最下沿到本行字体的最下沿

文本属性总结

属性表示注意点
color文本颜色通常用十六进制，而且是简写形式#fff
text-align文本对齐设定文本的水平对齐方式
text-indent文本缩进通常用于首行缩进2字符，text-indent: 2em;
text-decoration文本修饰添加下划线 underline，取消下划线 none
line-height行高控制行与行之间的距离
